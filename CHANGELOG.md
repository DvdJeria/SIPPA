# Changelog

Todos los cambios importantes del proyecto serán documentados en este archivo.

El formato está basado en **Keep a Changelog**:
https://keepachangelog.com/es-ES/1.1.0/

y este proyecto utiliza Versionado Semántico:
https://semver.org/lang/es/

---

## [0.1.0] - 2026-06-26

### Agregado

- Inicio del proyecto SIPPA.
- Sistema de autenticación con Supabase.
- Gestión de Ingredientes.
- Gestión de Unidades de Medida.
- Generación de Cotizaciones.
- Conversión de Cotizaciones a Pedidos.
- Gestión de Clientes.
- Integración con SQLite para funcionamiento offline.
- Sincronización inicial con Supabase.
- Publicación de la primera APK mediante GitHub Releases.

### Mejorado

- Refactorización del módulo de Cotizaciones.
- Mejoras en el cálculo de costos.
- Optimización de consultas SQLite.

### Corregido

- Corrección del problema de sincronización de Ingredientes en Android mediante `forceSync`.
- Corrección de diversos errores de estabilidad.
